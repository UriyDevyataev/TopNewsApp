# Приложение написано в рамках тестового задания, в самом начале карьеры. Поэтому не отражает уровень текущих навыков

# TopNewsApp

## Functions

- Display the latest 20 news in each category
- Caching news

## Technology stack

- Realm (for save data)
- SnapKit
- Using Open API https://newsapi.org
